# Outreach Playbook — LinkedIn DM & Connection Note

Untuk Mode Outreach di skill linkedin-beacon. Scope: pitching jasa digital
marketing (paid ads, SEO/GEO, konten) untuk personal brand Lawrence Philemon.
Asuransi tidak pernah dipitch lewat LinkedIn. Semua output draft only —
pengiriman selalu manual oleh Lawrence.

---

## Konstanta Platform (per Juli 2026 — verifikasi berkala, angka bisa berubah)

| Item | Batas | Konsekuensi kerja |
|---|---|---|
| Connection note | 200 karakter (free) / 300 (Premium, Sales Nav) | standar kerja: ≤200 termasuk spasi, aman di semua tier; target draft ≤180 sebagai buffer nama panjang |
| Jatah noted invite akun free | ±5 per bulan | note = amunisi langka; hanya untuk target prioritas, sisanya blank invite |
| Connection request mingguan | ±100/minggu baseline | bisa naik ke 150–200 kalau acceptance rate >30% dan profil sehat; acceptance <30% = throttle |
| Pending invites | tarik yang menggantung >2 minggu | tumpukan pending = sinyal spam ke LinkedIn |
| Laju pertumbuhan | ±3% dari network per minggu | 1.000 koneksi = ±30 invite/minggu |
| DM ke koneksi 1st degree | 8.000 karakter | tapi pesan <400 karakter punya response rate ±2x dibanding >1.000 — pendek menang |

## Note vs Blank Invite

Data antar studi konflik: satu dataset besar menemukan blank invite menang
(±55–68% vs 28–45% dengan note), studi lain menemukan sebaliknya. Konsensus
praktis yang dipakai skill ini:

- Note generik < blank invite < note personal genuine
- Note dipakai HANYA kalau ada angle spesifik yang tidak bisa di-copy ke
  orang lain (swap test)
- Target yang sudah engage dengan konten kamu → blank sering cukup,
  mereka sudah kenal nama kamu di feed
- Total stranger + angle genuine + target prioritas → note

## Sapaan & Register

- Sesama praktisi / selevel: "kamu", santai
- Owner / senior / calon klien formal: "pak/bu [nama]", kalimat tetap santai
  dan pendek — bukan bahasa surat
- Jangan pakai "Anda" — kaku
- lowercase-first, nol tanda seru, nol emoji

## Formula Connection Note

`[trigger spesifik — kenapa connect sekarang] + [relevansi kamu, satu frasa] + tanpa ask`

Pembuang karakter yang dilarang: "semoga pesan ini...", "perkenalkan, saya
dari...", "izin connect", flattery generik ("kontennya sangat menginspirasi").

### Template per skenario (isi placeholder, hitung ulang, wajib ≤200)

**1 — Habis engage konten mereka**
> halo [nama], baca post kamu soal [topik] — poin [detail spesifik] jarang
> dibahas sejujur itu. saya praktisi paid ads & GEO, senang connect sesama
> praktisi.

**2 — ICP cold, ada observasi bisnis**
> halo pak [nama], lihat [brand] lagi aktif di [channel]. saya bantu
> beberapa brand [industri] di area yang sama — connect dulu, semoga
> insight-nya kepakai.

**3 — Mutual connection / komunitas**
> halo [nama], kita satu circle di [komunitas / mutual]. saya di paid ads
> & GEO — senang nambah koneksi yang relevan.

**4 — Post event / webinar**
> halo [nama], ikut sesi [event] juga tadi — bagian [topik] menarik. saya
> praktisi [bidang], senang lanjut diskusinya.

**5 — Blank invite**
Tanpa note. Dipakai saat tidak ada angle genuine, atau jatah note bulan ini
lebih baik disimpan untuk target yang lebih prioritas.

## Sequence A — Inbound Hangat (dari konten, Fase 2 beacon)

Trigger: komentar berbobot, DM masuk, atau save + profile visit berulang.
Mereka datang sendiri — tidak perlu 3 tahap.

**Pesan 1:** jawab substansi komentar/pertanyaan mereka + satu pertanyaan
tentang situasi mereka.

**Pesan 2 (kalau problem terkonfirmasi):** diagnosis satu baris + bukti
angka + micro-CTA wedge.

## Sequence B — Cold-Warm (baru connect, belum pernah chat)

**Pesan 1 — opener tanpa pitch** (target <300 karakter):
> terima kasih sudah connect, [nama]. sempat lihat [aset: website / iklan /
> konten mereka] — [observasi spesifik]. penasaran, [pertanyaan genuine yang
> relevan dengan role mereka]?

Aturan pesan 1: nol link, nol penawaran, nol lampiran.

**Pesan 2 — value drop** (setelah ada reply):
> btw waktu buka [aset], saya notice [temuan spesifik + angka kalau ada].
> bukan pitch — kebiasaan auditor, sayang kalau tidak disampaikan.
> [satu saran kecil yang bisa langsung mereka pakai]

**Pesan 3 — pitch** (paling cepat di sini):
> kalau [problem] memang lagi jadi prioritas, saya bisa siapkan [wedge] —
> 3 temuan paling berdampak, 15 menit review bareng. kalau cocok baru kita
> bicara scope. worth it kalau saya kirim minggu ini?

Anatomy pitch: trigger → diagnosis satu baris → bukti (angka, bukan klaim)
→ satu micro-CTA.

## Wedge Offers (pintu masuk, bukan proposal full)

| Wedge | Untuk target | Bukti pendukung |
|---|---|---|
| Mini GEO audit — 3 temuan visibilitas brand di AI | brand yang belum muncul atau salah muncul di ChatGPT/Perplexity/Gemini | case study Gwenchana (sudah publik, boleh named) |
| Waste-scan Google Ads — budget tanpa konversi | bisnis yang sedang jalan ads | temuan conversion tracking discrepancy (anonim) |
| Quick-scan creative fatigue Meta | e-commerce / D2C aktif di Meta | framework fatigue (frequency, CTR decline, CPM inflation) |

Jangan pernah buka dengan "full service digital marketing" — terlalu besar
untuk DM. Wedge dulu, scope belakangan.

## Objection Quick-Reframes (validasi → reframe → langkah kecil)

| Objection | Validasi | Reframe | Langkah kecil |
|---|---|---|---|
| "sudah pakai agency" | bagus, berarti channel-nya sudah serius | audit pihak ketiga justru paling berguna saat sudah ada yang jalan — pembanding netral | satu temuan spesifik, silakan bawa ke agency-nya |
| "belum ada budget" | masuk akal, apalagi kondisi sekarang | sering kali budget-nya sudah ada, cuma bocor di waste | mulai dari temuan waste dulu, keputusan belakangan |
| "kirim proposal saja" | siap | proposal tanpa data = tebakan | 15 menit atau akses read-only dulu supaya angkanya nyata |

## Anti-Patterns

| Jangan | Kenapa |
|---|---|
| "perkenalkan, saya dari [agency] yang bergerak di bidang..." | opener CV = auto-ignore |
| pitch di dalam connection note | note = konteks, bukan jualan |
| "just following up" / "up" tanpa value baru | dibaca sebagai desperate |
| wall of text di pesan pertama | DM dibaca di layar HP; 3–5 baris maksimal |
| flattery generik | tidak spesifik = tidak dibaca |
| link booking/calendly di pesan pertama | ask terlalu besar terlalu cepat |
| satu template diblast tanpa edit | gagal swap test; acceptance rate turun = akun di-throttle |

## Follow-up Rules

- Maksimal 2 follow-up tanpa respon; jeda 4–7 hari
- Setiap follow-up bawa value baru: data point tambahan, temuan lain,
  artikel relevan milik sendiri — bukan pengingat kosong
- Setelah 2 follow-up tanpa respon → berhenti. Energi pindah ke konten;
  mereka tetap melihat kamu di feed, dan itu kerja Fase 1

## Self-Check Outreach (semua wajib lolos sebelum deliver)

- [ ] Connection note ≤200 karakter — hitung dan tulis angkanya di output
- [ ] Swap test lolos: minimal satu detail yang hanya berlaku untuk target ini
- [ ] Sequence B: pitch tidak muncul sebelum pesan ke-3
- [ ] Satu CTA per pesan, kecil, spesifik
- [ ] Nol tanda seru, nol emoji, nol forbidden words
- [ ] Setiap klaim ada angkanya, atau tidak ditulis
- [ ] Tidak ada janji hasil atau jaminan apapun
- [ ] Register sapaan sesuai target (kamu vs pak/bu)
- [ ] Draft only — tidak ada pengiriman atau scheduling otomatis
